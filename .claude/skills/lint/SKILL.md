# /lint - 린트 실행

ESLint를 실행하고 문제를 자동으로 수정합니다.

## 실행 단계

1. **린트 실행**
   ```bash
   npm run lint
   ```

2. **문제 분석**
   - 오류 (error) 분류
   - 경고 (warning) 분류
   - 파일별 문제 그룹화

3. **자동 수정**
   ```bash
   npm run lint -- --fix
   ```

4. **수동 수정 필요 항목**
   - 자동 수정 불가 항목 목록화
   - 수정 방법 제안

## 출력

```
✅ 린트 통과
- 검사 파일: X개
- 수정된 문제: X개

또는

⚠️ 린트 경고
- 오류: X개
- 경고: X개
- [문제 목록 및 수정 제안]
```
