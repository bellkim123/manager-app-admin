# /test - 테스트 실행

테스트를 실행하고 결과를 분석합니다.

## 실행 단계

1. **테스트 실행**
   ```bash
   npm test
   # 또는 Jest가 설정된 경우
   npx jest --passWithNoTests
   ```

2. **결과 분석**
   - 통과/실패 테스트 수
   - 실패 원인 분석
   - 커버리지 확인 (설정된 경우)

3. **실패 테스트 수정**
   - 테스트 코드 문제 시 수정
   - 구현 코드 문제 시 수정 제안

## 출력

```
✅ 테스트 통과
- 총 테스트: X개
- 통과: X개
- 스킵: X개
- 커버리지: X%

또는

❌ 테스트 실패
- 실패: X개
- [실패 테스트 목록 및 원인]
```
