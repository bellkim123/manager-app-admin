# /build - 빌드 실행

프로젝트를 빌드하고 오류를 자동으로 수정합니다.

## 실행 단계

1. **빌드 실행**
   ```bash
   npm run build
   ```

2. **오류 분석**
   - TypeScript 타입 오류 확인
   - ESLint 오류 확인
   - 빌드 경고 분석

3. **자동 수정**
   - 타입 오류 수정
   - import 경로 수정
   - 누락된 의존성 해결

4. **검증**
   - 수정 후 재빌드
   - 성공 시 완료 보고

## 출력

```
✅ 빌드 성공
- 빌드 시간: X초
- 번들 크기: XXkB

또는

❌ 빌드 실패
- 오류 X개 발견
- [오류 목록 및 수정 내역]
```
